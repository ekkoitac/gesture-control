# Iteration Record: v0.2 Cursor Smoothness Implementation

## Goal

- Complete the active `v0.2-cursor-smoothness` TODO.
- Move index-finger cursor control from absolute screen positioning to relative normalized movement.
- Keep activation, pause, tracking-loss, pinch, wave, shortcut, and dry-run safety behavior unchanged.

## Runtime Changes

- Bumped package metadata to `0.2.0`.
- Added an explicit cursor movement mode contract: `absolute` and `relative`.
- Defaulted gesture config to `cursor_mode: relative`.
- Added cursor tuning config for fast smoothing, jitter floor, and max normalized step.
- Updated `GestureEngine` cursor handling:
  - first pointing frame seeds the baseline with zero movement,
  - tiny raw motion below the jitter floor emits zero delta,
  - deliberate motion is smoothed with speed-adaptive alpha,
  - oversized normalized deltas are clamped,
  - cursor baseline resets after pointing loss, no-hand, or pause.
- Updated `ActionMapper` to emit cursor commands with explicit `mode` and normalized `delta`.
- Updated `MacOSInputBackend` to apply relative cursor deltas from the current pointer position, flip x for user perspective, preserve y direction, and clamp to screen bounds.
